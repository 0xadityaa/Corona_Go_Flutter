# Corona Go
This is a Corona Virus data tracking application made using Flutter which gives realtime stats of all the countries.

# This app includes:
API calls<br/>
Dynamic Theme<br/>
Clean UI<br/>

# API used:
https://corona.lmao.ninja/v3/covid-19/countries
